# Digital Accessibility Guidebook

## Audit
### Manual Testing
* Keyboard Accessible
  * Skip Link
  * Tab-through
  * Focus visible
* Alternative text Content
  * Alt-text Image/ video
  * Custom controls 
  * Screen reader focus
* Page Structure
  * Headig order
  * Land mark elements
* Color and Contrast
